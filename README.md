# README

# Project Setup
-run "bundle install"
-run "rails db:setup"
-In client directory run "yarn install"
-In root directory run "foreman start"
	-starts rails server on port 3000 and react server on port 3001

# Tech
-Rails API
-React scaffold using create-react-app
